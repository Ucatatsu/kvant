### Техническое
- Обновлена функция `saveGroupMessage()` для поддержки `replyToId`

## [1B1a1] - 2025-12-31

### Новая фича: Быстрые ответы на сообщения свайпами и двойным кликом
- **Свайп для ответа на мобильных устройствах**:
  - Свайп влево для исходящих сообщений (отправленных вами)
  - Свайп вправо для входящих сообщений (полученных от других)
  - Визуальная обратная связь с иконкой ↩️ и градиентной подсветкой
  - Плавная анимация с возвратом в исходное положение
- **Двойной клик для ответа на десктопе** - работает для всех сообщений
- **Клавиатурное сокращение** - нажмите R на фокусированном сообщении
- **Настройки пользователя**:
  - Включение/отключение свайпов для ответов
  - Тактильная обратная связь (вибрация на мобильных)
  - Звуковая обратная связь при активации ответа
- **Accessibility улучшения** - поддержка клавиатурной навигации и ARIA-атрибуты
- **Улучшенная анимация** - пульсация при активации ответа с градиентной подсветкой
- **Адаптивный дизайн** - оптимизировано для мобильных и десктопных устройств
- Добавлена SVG иконка `/assets/quick-reply.svg` для быстрых ответов

### Техническое
- Новая функция `addReplyHandlers()` для обработки touch и click событий
- Функции `triggerReply()` и `resetMessagePosition()` для анимации
- CSS анимации: `.swiping`, `.reply-triggered`, `@keyframes replyPulse`
- Поддержка CSS переменных `--reply-indicator-opacity` для плавности
- Оптимизация производительности с `passive: true/false` для touch событий
- Предотвращение конфликтов с контекстным меню и скроллом
- Интеграция с системой настроек пользователя
- Генерация звуковых сигналов через Web Audio API
- Поддержка Vibration API для тактильной обратной связи

- Интеграция lottie-web 5.12.2 для анимаций
- Интеграция pako 2.1.0 для декомпрессии gzip
- Новый тип сообщения `sticker` в базе данных
- Стикеры сохраняются как JSON в поле text
- Поддержка стикеров в push-уведомлениях (🎭 Стикер)
